Generic Copay Help

Select the copayment amount you pay for generic medications.

If the amount is not listed select OTHER and enter the amount.

If the amount varies each time select VARIES.

If you are not sure about your generic copayment select NOT SURE.

Brand Copay Help

If you pay the same copayment amount for all brand name medications select YES.

If you do not know your brand copayment amount or if it varies each time select NO.
